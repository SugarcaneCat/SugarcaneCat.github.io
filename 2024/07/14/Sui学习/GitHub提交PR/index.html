# 用 GIt 实现 GitHub 的 Pull requests 提交

参考作者：Elemen_7

# 安装 Git 工具

这一步有很多教程，百度一下即可。

# 开始

  1. fork 目标仓库到自己的仓库地址中，其他默认点击 creat 即可

# 复制远程仓库地址到本地文件夹中

  1. 复制仓库地址，即 fork 后自己的仓库的项目地址

  2. 随后在本地任意文件夹 (最好空白文件夹) 中，按住键盘 SHIFT + 右键，选择如图所示

  3. 在打开的 git bash 中输入：

    git clone （刚刚复制的仓库地址）

  4. 这个时候你本地会出现一个同名文件夹，这就是克隆成功了 ps: 这一步可能对网络要求较高，建议开启代理。

# 选择多线程工作，需要创造分支

  1. 创建分支

    git checkout -b 分支名

  2. 删除分支，先切换到非删除分支，再进行删除操作

    git branch -d 删除的分支名

  3. 查看所有分支
    git branch -a

# 推送分支到远程仓库

  1. 将修改文件加入缓存区

    git add .

  2. 提交
    git commit -m"修改信息"

  3. 推送
    git push --set-upstream origin 分支名

# 回到 GitHub，提交 PR

  1. 在 GitHub 页面刷新后，将分支切换为刚刚提交的分支，准备创建 PR，如图

  2. 创建之后，在文件栏填写相应的命名要求即可.
